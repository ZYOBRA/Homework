# Инсрукция для работы с Markdown

## Выделение текста
Чтобы выделить текст курсивом нужно обрамить его звездочками (*).
*Например вот так*.

Чтобы выделить полужирным, необходимо обрамить двойными звездочками (**).
Например **Вот так**.
